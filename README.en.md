#### Installation

1.  gcc/g++ version 9.3.0
2.  cmake version mini 3.15
3.  mkdir build
4.  cd build
5.  cmake ../restool_standard
6.  make
7.  compile result restool binary

#### Instructions

1.  restool -v show version.
2.  restool show help.
